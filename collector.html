Collector

Keyboard Shortcuts
CapsLock+;Collect selected text into the active list
Shift+CapsLock+;Create a new list named from clipboard contents
CapsLock+]Paste all items (space-separated) into current app
CapsLock+BackspaceToggle this viewer window
How to Use
1. Create a list — click + New List above, or copy an address or any text and press Shift+CapsLock+; to name a list from clipboard.
2. Collect items — highlight text (e.g. an MLS number) anywhere and press CapsLock+;. It gets added to the active list.
3. Review — open this viewer with CapsLock+Backspace to see, reorder, or remove items. You can also type/paste items manually above.
4. Paste — press CapsLock+] to paste all items space-separated, or click Copy List to copy to clipboard.
5. Manage — switch between lists via tabs, rename with Rename, or clear/delete as needed.
